# Vue3-Express-MongoDB-Demo

## project-api
```
cd project-api
npm start
```

## project-web
```
cd project-web
npm install
```

## The technology stack of demo is as follows:
- Vite + Vue3 + TypeScript
- Element Plus + Font Awesome
- Express + Nodejs + JavaScript
- MongoDB + Mongoose
